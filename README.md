# Titanic Survival Analysis Dashboard

## Project Overview

This project analyzes the Titanic dataset using SQL and Power BI to understand passenger demographics, ticket classes, and survival patterns.

## Tools Used

* Power BI
* SQL (MySQL)
* Excel
* Data Visualization

## Key Features

* Interactive dashboard with slicers
* Passenger distribution analysis
* Gender-wise analysis
* Age-group analysis
* Ticket-class distribution
* Survival analysis

## SQL Analysis

Performed data analysis using:

* COUNT()
* GROUP BY
* AVG()
* ORDER BY
* Aggregation functions

## Dashboard Insights

* Female passengers had a higher survival rate than male passengers.
* First-class passengers had better survival chances.
* Most passengers belonged to third class.
* Average passenger age was approximately 30 years.
